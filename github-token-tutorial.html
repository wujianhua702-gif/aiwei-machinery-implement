🔑 GitHub Classic Token 生成教程

按照以下步骤生成一个有 repo 权限的 Personal Access Token (Classic)

1
登录 GitHub，进入 Token 设置页
打开浏览器，访问 github.com/settings/tokens
确保你已经登录了 GitHub 账号。
2
点击生成按钮
在 Token 设置页面，点击 Generate new token 按钮，
然后从下拉菜单中选择 Generate new token (classic)。
🔄 [ 示意图 ] — 点击 "Generate new token" → "Generate new token (classic)"
3
填写 Token 信息
Note：给 Token 起个名字，比如 codex-kingdee
Expiration：建议选 No expiration，免得过期后忘了续
Select scopes：勾选 repo
💡 必选权限：
✅ repo — 完全控制私有仓库（包含 public_repo）
其他权限按需勾选，创建仓库只需要 repo
4
勾选 repo 范围
在 Scopes 区域，找到 repo 并勾选它。
勾选后会自动选中下属所有子权限。
✅ [ 示意图 ] — Scopes 列表中找到 repo，勾选
5
生成并复制 Token
拉到页面底部，点击 Generate token 按钮。
生成后你会看到一个绿色的长字符串——立即复制它！
这个 Token 只显示一次，关闭页面后就再也看不到了。
⚠️
务必保存好 Token
复制后粘贴到安全的地方（比如密码管理器）
然后发给我，我帮你创建仓库
